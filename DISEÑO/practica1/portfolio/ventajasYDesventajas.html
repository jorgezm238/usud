Ventajas y desventajas de usar Node js

Debido a la buena arquitectura de Node js se ha convertido en uno de los entornos de ejecución más utilizados del mercado y lo que más crecimiento están recibiendo.

Ventajas de Node js:

Entre las ventajas más relevantes de utilizar Node js encontramos:

  1. Escalable: es un entono perfecto para construir aplicaciones en red escalables con un alto nivel de rendimiento.
  2. Rendimiento y eficacia: la ejecución de procesos sin bloqueo y la utilización de menos recursos hace que sea un entorno rápido y muy eficaz.
  3. Facilidad: Utiliza javascript un lenguaje muy bien estructurado y relativamente fácil de aprender.
  4. Open source: no es necesario el tener ninguún tipo de licencia para su uso
  5. Comunidad y soporte: plataformas como github han incitado al crecimiento de node, proporcionando una gran documentación

Desventajas de Node js

Pero no solo tiene ventajas si no que también tiene algún inconveniente a tener en cuenta como:

  1. Curva de aprendizaje: requiere muchas lineas de código para uso lo que dificulta el aprendizaje del mismo
  2. Compatibilidad: necesita un hosting especializado
  3. Módulos en desarrollo: escasos modulos y dependencias en comparación con otros sistemas
  4. Documentación: mucha documentación no está traducida
